## Installation

Instructions on how to install and run the project.

1. Clone the repository:

   ```bash
   https://github.com/Yeezuu-web/node-mongo-boilerplate-jwt.git
   ```

2. Install dependencies:

   ```bash
   cd project-name
   npm install
   ```

3. Start the application:
   ```bash
   npm start
   ```

## Usage
